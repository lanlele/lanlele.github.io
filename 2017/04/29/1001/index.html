淘宝按销量维度批量抓取商品的价格、销量和标题等

发表于 2017-04-29 | 分类于 1000 需求管理

1.准备工作

1.1安装谷歌chrome浏览器

下载链接：http://rj.baidu.com/soft/detail/14744.html

1.2下载Web Scraper插件（crx文件）

下载链接： http://pan.baidu.com/s/1dE1AxVz 密码：n3cy

1.3手动安装插件

安装教程：http://jingyan.baidu.com/article/e5c39bf56286ae39d6603374.html

1.4使用插件

打开谷歌控制台（按快捷键F12），选中web scraper即可使用web scraper

2.操作步骤

3步实现商品数据抓取：

2.1创建抓取事件

创建步骤：

  1. web scraper
  2. Create new sitemap
  3. Import sitemap
  4. Sitemap JSON区域输入代码
  5. 输入站点名（自定义）
  6. 点击导入按钮

2.2点击scrape设置抓取

2.3设置抓取时间间隔，开始抓取数据

3.如何抓取多页的商品信息

代码默认仅抓取第一页的商品，若大家需抓取更多页的商品，仅需按照以下操作修改代码中对应的商品链接即可，以boku为例。

默认抓取的第一页:

https://bokuts.tmall.com/category.htm?spm=a1z10.5-b-s.w4011-15421479631&scene=taobao_shop&pageNo=1

设置为抓取第二页:

https://bokuts.tmall.com/category.htm?spm=a1z10.5-b-s.w4011-15421479631&scene=taobao_shop&pageNo=2 |

设置为抓取1到10页:

https://bokuts.tmall.com/category.htm?spm=a1z10.5-b-s.w4011-15421479631&scene=taobao_shop&pageNo=[1-10]

从上面3个链接可看出，仅需修改地址后的数字，即可自定义抓取相关页面的商品，可多人合作抓取店铺内商品的信息再汇总，设置办法如下：
