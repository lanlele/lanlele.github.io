用户体验要素

发表于 2017-05-07 | 分类于 2000 产品设计

用户体验可以转换成一个简单清晰的模型：可分为5个层次10个要素，能系统化地诠释“用户”、“商业”和“技术”三者之间错综复杂的关系。

  • 战略层：成功的用户体验，其基础是一个被明确表达的“战略”。当我们能越清楚地表达我们想要什么，以及确切地知道其他人想要从我们这里得到什么时，我们就能越精确地满足双方的需求。
  • 范围层：带着“我们想要什么”、“我们的用户想要什么”的明确认识，我们才能把用户需求和产品目标转变成产品应该提供给用户什么样的内容和功能。
  • 结构层：在定义好用户需求并排列好优先级别之后，我们清楚地知道最终产品将会包括什么特性。结构层将为网站创建一个概念结构，串联范围层确定的功能和内容（即分散的片段）组成一个整体。
  • 框架层：在充满概念的结构层中形成了大量的需求，在框架层，需进一步提炼这些结构，确定很详细的界面外观、导航和信息设计，让晦涩的结构变得更实在。
  • 表现层：感知设计。这里，内容、功能和美学汇集到一起来产生一个最终设计，完成其他四个层面的所有目标，并同时满足用户的感官感受。

内容来源于《用户体验要素》Jesse James Garrett著
