面试招聘怎能不懂BEI

发表于 2017-03-13 | 分类于 招聘面试 |

1.概念

1.1什么是行为事件访谈法BEI

人以往表现出来的行为方式是人未来行为最可靠的预测指标！

BEI（Behavioral Event Interview）：它是通过被访谈者进行反复提问，收集被访谈者在过去有代表性的事件中的具体行为和心理活动的详细信息，进而通过对所收集信息的对比分析，发现杰出者普遍具备而胜任者普遍缺乏的个性特质——即素质。

1.2什么是素质冰山模型

1.2.1表象

技能：指结构化地运用知识完成某项具体工作的能力，即对某一特定领悟所需技术与知识的掌握情况。

知识：指个人在某一特定领域拥有的事实型与经验型信息。

社会角色：指一个人留给大家的形象。

自我形象：是一个人对自己的看法，即内在自己认同的本我。

1.2.2潜层

品质：指个性、身体特征对环境与各种信息所表现出来的持续而稳定的行为特征。品质与动机可以预测个人在长期无人监督下的工作伏态。

动机：指在一个特定领域的自然而持续的想法和偏好（如成就、亲和、影响力），它们将驱动，引导和决定个人的外在行动。

1.2.3绩效

绩效水平由素质的五个层次的综台因素决定，即有易于感知的知识、技能与行为，又有难以被挖掘与感知的潜能。

更进一步的，“水面上”知识与技能等仅仅是冰山的一个小角，“水面下”的更宏大的潜在索质，对绩效起到更大的决定作用。

1.3如何去实现BEI—STAR工具

S（Situation）:那是一个怎么样的情境？什么样的因素导致这样的情境？在这个情境中有谁参与？

T（Task）：您面临的主要任务是什么？为了达到什么样的目标？

A（Action）：在那样的情境下，您当时心中的想法、感觉和想要采取的行为是什么？

R（Result）：最后的结果是什么，过程中又发生了什么？

2.访谈BEI

2.1掌握面试者当前的状况

掌握面试者当前的工作状况（对有工作经验者），如你所在的岗位名称是什么；你向谁汇报；你在这个岗位上工作了多久；你的主要职责是什么；你的时间是怎么分配的。

了解更多关于他工作职责和任务的信息，如你可以解释一下什么叫做带领一个团队吗。

在这个阶段，你就可以让被访谈者感受到你需要更多细节方面的信息。如你在这里说的管理是什么意思呢；你可以给我讲一个例子，来解释你怎么带领你的团队去工作的吗；在你说制定计划事，通常需要涉及到哪些事情呢。

2.2访谈中要发现的四点FACT

Feeling：当事情发生时，你是什么感受呢？

Action：你当时说了什么？你当时做了什么？

Context：你当时面对的情况是怎么样的？你的角色是什么？还有谁在事情当中？事情得结果是怎么样的？

Thinking：事情发生的那一刻，你正在想什么？你当时想什么呢？

2.3识别面试者的有效与无效对话

有效数据

  • 主角是“我”
  • 特定的行为
  • 面试者自发的想法
  • 事件发生当时的想法、行动和感受
  • 具体的行动

无效数据

  • 主角是“我们”
  • 一般的情况，如“我通常会这么做”
  • 面试官引导被面试者做出的回答
  • 对过去时间的现在感受
  • 模糊的总结

2.4问适当的问题

尽量避免引导被访谈者进入抽象化的思考。如为什么您这么做？更恰当的问题是：您当时的想法是什么？

避免问引导性的问题。如所以，您极力要影响他改变态度，是吗？更恰当的问题是：您当时对他说了些什么？

避免出现“我们”。如我们当时做出了决策？更恰当的问题是：我们是指谁？具体到您，当时做了什么？

避免“假设性”表述。如如果当时他不跟我们去的话，我将……？更恰当的问题是：能否给我一个实例，在这种情况下，您是怎么做的？

避免一般性、模糊性表述。如一般来说，……，通常，我会……。更恰当的问题是：您当时具体做了什么？

练习示例

  • 责任心：当你的工作因为种种原因进展不下去时，你是怎么处理的？
  • 团队合作：请举一个你配合他人将工作完成的例子。
  • 创新精神：你曾经尝试用过新的方法来完成你的工作或学习吗？能详细描述一下吗？
  • 积极进取：请讲一个你的故事证明你喜欢做一些挑战性的工作或参与挑战性的活动。
  • 团队领导：你是否领导过一个团队呢？能具体描述其中的职责吗？
  • 学习精神：你可以举个关于你自己不断学习，不断提高自己的例子吗？

2.5注意事项

避免成为以下角色：

  • 调研员：你管理多少员工？你工作有多久了？
  • 医师：所以在那种情况下你会……
  • 理论家：为什么你想那样做？
  • 算命先生：如果那样做，会……
  • 销售人员：难道你不觉得这个方法更好吗？

3.经验总结BEI

掌控访谈、挖掘事实、注意分辨、注意信息收集

本科访谈表：计划性、锐意创新、逻辑思维

专科访谈表：坚韧抗压、责任感、逻辑思维
